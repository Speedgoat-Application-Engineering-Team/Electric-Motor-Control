Speedgoat HIL Kit

The Speedgoat HIL Kit with a Texas Instruments (TI) microcontroller is an excellent starting point to HIL testing using Simulink and Speedgoat hardware.
The kit contains a TI microcontroller and interface to the Speedgoat IO397 FPGA I/O Module (see image below).

Connecting the Demo

Prepare the demo hardware with the following three steps:
  • Connect HIL Kit to the Speedgoat target computer. This can be done by connecting the HIL Kit using to the IO397 I/O module either by:
1) Using the M12 cables provided or,
2) Directly plugging the HIL hardware in the M12 connectors of the speedgoat target computer.

Kit Power Options

  • Recommended option: While the HIL demo requires a 5 Vdc power supply, the Speedgoat target compuer includes a 5 Vdc power supply. To provide power to the demo set the power switch labelled Speedgoat 5V Supply to the ON position to use the Baseline power supply. No external power supply or USB cable should be connected to the HIL Kit.
  • Alternative option: By setting the power switch labelled Speedgoat 5V Supply to the OFF position, it is possible to power the HIL Kit either by (1) using the USB cable provided or (2) using an external 5Vdc regulated power supply with a 5.5mm/2.1mm 5 Vdc Barrel Jack connector (not included with HIL Kit). The JP2 switch next to the USB connector should be set according to the external power supply used.

Power-On Kit

To start using the HIL Kit, turn ON the switch on the top PCB (marked as Speedgoat PCB Switch in the image above). The TI ControlCard with the TI microcontroller will power ON.
  • LED1 is solid green and LED3 is blinking on the TI ControlCard: it means that a motor controller is already deplyoed and running in the TI microcontroller;
  • LEDs do not blink: load a motor controller into the TI microcontroller by following the exercise Perform HIL Testing
